# Arduino Pet Feeder

Automatic pet feeder with joystick trigger, dual servos, and buzzer notifications.
Link: https://wokwi.com/projects/427181619999669249
## Tech
- Arduino Mega
- Servo motor (positional)
- Servo motor (continuous rotation)
- Joystick
- Buzzer + LEDs

## Features
- Open/close door servo
- Dispense food with rotation servo
- Buzzer + LED indicators for feedback
- Joystick to trigger feeding sequence
